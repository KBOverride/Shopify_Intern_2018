********
Overview
********

I have one solution to the challenge and another solution for the extra challenge. Both solutions obtain a list
of nodes from the API and build the appropiate menus. The output is in JSON format and is shown through the
console of the browser.

**************
Challenge Link 
**************

https://backend-challenge-summer-2018.herokuapp.com/

**************************
Programs and Software used
**************************

- JavaScript
- HTML
- Google Chrome
- "Allow-Control-Allow-Origin: *" Chrome extenstion
- Graphlib (data structures for graphs)

************
Requirements
************

- Google Chrome (To run flags that allow you to request any site with ajax from any source)
- "Allow-Control-Allow-Origin: *" Chrome extenstion (see option below)
	- Alternative: Add these flags under properties of Google Chrome: 
	--allow-file-access-from-files --allow-file-access --allow-cross-origin-auth-prompt
	chrome --disable-web-security

***************************
How to Run Normal Challenge
***************************

These are the steps I took to run the program.

1. Install "Allow-Control-Allow-Origin: *" chrome extension (https://chrome.google.com/webstore/detail/allow-control-allow-origi/nlfbmbojpeacfghkpbjhddihlkkiljbi) 

OR 

1. Add chrome flags under properties/target of Google Chrome (may not work)
	chrome --disable-web-security
	--allow-file-access-from-files --allow-file-access --allow-cross-origin-auth-prompt

2. Open HTML file called "challenge.html"

3. Look at web page or console for output

**************************
How to Run Extra Challenge
**************************

Instructions are same as above except that in Step 2, you open the html file called "extraChallenge.html".
